# 🎯 CTR — Sistema de Controle de Munição
## Clube de Tiro do Recôncavo

Sistema web completo para controle de estoque de munição, cadastro de clientes e relatório de habitualidade.

---

## 🌐 Acesso Online
**Link do sistema:** https://SEU-USUARIO.github.io/ctr-municao

---

## 📱 Como usar no celular
1. Acesse o link acima pelo Chrome ou Safari
2. Toque nos **3 pontinhos** (Android) ou **compartilhar** (iPhone)
3. Selecione **"Adicionar à tela inicial"**
4. Pronto — fica como um app na tela inicial!

---

## 🚀 Como publicar no GitHub (passo a passo)

### 1. Criar conta
- Acesse **github.com** e crie uma conta gratuita
- Use o e-mail do clube

### 2. Criar repositório
- Clique em **"New repository"**
- Nome: `ctr-municao`
- Marque **"Public"**
- Clique **"Create repository"**

### 3. Fazer upload dos arquivos
- Na página do repositório, clique **"uploading an existing file"**
- Arraste os arquivos: `index.html` e `README.md`
- Clique **"Commit changes"**

### 4. Ativar GitHub Pages
- Vá em **Settings** (engrenagem)
- No menu lateral: **Pages**
- Em "Branch": selecione **main**
- Clique **Save**
- Aguarde 2 minutos

### 5. Acessar
- Seu link será: `https://SEU-USUARIO.github.io/ctr-municao`
- Compartilhe com a equipe do clube!

## 📋 Funcionalidades
- 📊 Dashboard com totais em tempo real
- 🔫 Cadastro livre de munições (qualquer calibre)
- 📦 Controle de estoque com alertas de mínimo
- 🎯 Registro de retiradas com desconto automático
- 👤 Cadastro de clientes com CPF e nº de filiado
- 📅 Histórico organizado por mês (a partir de Maio/2025)
- 🏆 Relatório de Habitualidade para lançamento no sistema do clube
- ☁️ Sincronização com Google Sheets
